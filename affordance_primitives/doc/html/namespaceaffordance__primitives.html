Eigen::MatrixXd affordance_primitives::getAdjointMatrix ( const Eigen::Isometry3d &  transform)

Converts a transformation into an adjoint matrix for transforming twists and wrenches.

Usage: twist_in_A = Adjoint(tf_from_A_to_B) * twist_in_B

When using this adjoint, it assumes the twist is a 6x1 vector with linear on top [linear ; angular]. This follows from the way tf2_eigen converts between twist messages and Eigen vectors, but is opposite from certain math notation

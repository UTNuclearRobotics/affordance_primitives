Converts a transformation into an adjoint matrix for transforming twists and wrenches.

Usage: twist_in_A = Adjoint(tf_from_A_to_B) * twist_in_B

When using this adjoint, it assumes the twist is a 6x1 vector with angular on top [angular ; linear]. The tf2_eigen library does not convert Twist messages to Eigen Vectors this way, so use the functions included here for converting between these data types
